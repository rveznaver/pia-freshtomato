# pia-freshtomato
Script to set up PIA wireguard on FreshTomato

Example setup:
```
scp -O piascript.sh root@<ROUTER_IP>:
pia_user='<REDACTED>' pia_pass='<REDACTED>' ./piascript.sh
```
